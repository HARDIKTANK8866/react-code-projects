# React + Vite Password Generator

This project provides a minimal setup to build a password generator using React and Vite.

## Features

- Generate random passwords based on user-defined criteria.
- Demonstrates the use of React hooks such as `useState`, `useEffect`, `useRef`, and `useCallback`.
